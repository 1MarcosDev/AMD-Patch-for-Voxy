# AMD Patch for Voxy

Fixes rendering issues on AMD RX500 series cards for Voxy by patching the screen space shader at runtime.

## Author
Marquinhos0550

## Building
This project requires the Voxy source code to be present in a `voxy-amd-patch` subdirectory to compile against.
The `voxy-amd-patch` directory is not included in this repository due to licensing restrictions.
